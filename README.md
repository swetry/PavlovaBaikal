# Безопасный SQL Runner
Отборочное задание для ИТ-школы КОД БАЙКАЛА 2026.

## Требования
- Python 3.10+
- PostgreSQL (локально или удалённо)
  
## Установка

Клонируйте репозиторий и установите зависимости:
```bash
git clone <repo-url>
cd safe-sql-runner
pip install -r requirements.txt
cp .env.example .env```
