Безопасный SQL Runner
Отборочное задание для ИТ-школы КОД БАЙКАЛА 2026.

Описание
Простой Python-скрипт, который безопасно подключается к PostgreSQL и позволяет выполнять только SELECT-запросы. Если в запросе отсутствует ограничение на количество строк, скрипт автоматически добавляет LIMIT 5. Любые попытки изменения данных (DELETE, UPDATE, INSERT и др.) блокируются.

Стек
Python 3.10+
psycopg2-binary
python-dotenv
Требования
Установленный Python 3.10 или выше.
Доступная база данных PostgreSQL (локально или удаленно).

Наличие собственного LIMIT
Если пользователь уже указал лимит, дублирование не происходит.

Ввод:

Копировать

Выполнится как есть: SELECT name FROM products LIMIT 3;

Тестирование перед отправкой
Перед коммитом убедитесь, что ваш код проходит следующие проверки:

Безопасность пакета: ввод SELECT 1; DELETE FROM test; должен выдать ошибку.
Работа LIMIT: ввод SELECT 1 LIMIT 10; не должен падать с ошибкой синтаксиса.
Интерактивный пароль: запустите скрипт после переименования или удаления файла .env, чтобы убедиться, что getpass() срабатывает корректно.
Стиль кода: проверьте отсутствие лишних импортов и соблюдение PEP 8.
